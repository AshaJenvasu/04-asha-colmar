# 🏛️ Colmar Academy - CSS Assessment

The project focuses on creating a responsive landing page for an educational institution using advanced CSS layout techniques.

---

## 🎯 Core Philosophy: "Box within a Box"
My primary approach to this layout was treating every element as a structured box. By utilizing **Flexbox** as the foundation, I created a primary container holding multiple nested flexboxes. This allowed for:
- Precision control over row and column arrangements.
- A clean, scalable, and modular code structure.
- Easier debugging and component management.

---

## 🚀 Key Features & Development Process

### 1. Pure CSS Mastery 🎨
While frameworks like Tailwind were options, I intentionally chose **Pure CSS** to strengthen my understanding of:
- Box Model & Layout behaviors.
- Complex spacing and alignment using `justify-content` and `align-items`.
- Percentages vs. Fixed widths for fluid layouts.

### 2. Multi-Layered Responsive Design 📱
Facing the challenge of responsiveness head-on, I implemented:
- **Media Queries**: Specifically targeting a `768px` breakpoint for tablets and mobile devices.
- **Mobile-First Approach**: Experimented with mobile-first logic in later sections to simplify code.
- **Adaptive Media**: Used the `<picture>` tag and source sets to serve optimized images for different screen sizes.

### 3. Systematic Workflow 🛠️
Developed the site from the top-down (Header to Footer) while maintaining a real-time feedback loop between **VS Code** and a **Live Preview** to ensure pixel-perfect results.
